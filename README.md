# Car Rental System (C++)

## Overview

The **Car Rental System** is a console-based application developed using **C++** that simulates a basic car rental management system. The program allows users to view available cars, rent cars, and return them through a menu-driven interface.

The project demonstrates the practical implementation of **Object Oriented Programming (OOP)** concepts and **file handling** in C++.



## Features

* View available cars
* Rent a car
* Return a rented car
* Store car data using file handling
* Menu-driven console interface
* Implementation using **classes, constructors, and inheritance**



## Technologies Used

* **C++**
* **Object Oriented Programming**
* **File Handling**
* **Standard C++ Libraries**



## How the System Works

1. The program loads car data from the **cars.txt** file.
2. A menu is displayed to the user with different options.
3. The user can:

   * View available cars
   * Rent a car
   * Return a car
4. Any changes made are stored back into the **cars.txt** file.
5. The program continues running until the user chooses to exit.



## How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/naikrumana6-create/Car-Rental-System-CPP.git
```

### 2. Navigate to the project folder

```
cd Car-Rental-System-CPP
```

### 3. Compile the program

```
g++ main.cpp -o car_rental
```

### 4. Run the program

```
./car_rental
```
